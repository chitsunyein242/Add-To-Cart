 🛒 Sneakers Shop – Add to Cart

A simple and responsive shopping cart project where users can browse sneakers, add them to the cart, adjust quantities, and see the total price. The cart is saved automatically in the browser using localStorage.

________________________________________
🚀 Features

🟢 Product Display

•    Shows sneaker image, title, and price

•    Each item has an Add to Cart button

🟢 Cart System

•    Add items to cart

•    Increase or decrease quantity

•    Remove items individually

•    Shows total item count

•    Shows total price

•    All updates happen instantly

🟢 Persistent Cart

•    Cart data is stored in localStorage

•    Page refresh does not reset the cart
________________________________________

🛠️ Technologies Used

•    HTML

•    CSS

•    JavaScript

•    localStorage
